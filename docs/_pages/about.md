## 안녕하세요! 👋

**성장**을 추구하는 개발자 PHSTD입니다.

### 🎯 블로그 소개

이 블로그는 제가 개발하면서 배운 것들, 경험한 것들, 그리고 생각한 것들을 기록하는 공간입니다.

### 💡 블로그 목적

- 📚 **학습 기록**: 새로 배운 기술과 개념들을 정리
- 🔄 **경험 공유**: 프로젝트 진행 과정과 문제 해결 경험
- 💭 **생각 정리**: 개발과 관련된 다양한 생각들
- 🤝 **지식 나눔**: 다른 개발자들과의 지식 공유

### 🛠️ 주요 관심사

- **웹 개발**: Frontend & Backend
- **게임 개발**: Unity
- **모바일 개발**: React Native, Flutter
- **데이터베이스**: MySQL, PostgreSQL, MongoDB
- **클라우드**: AWS, Docker
- **개발 도구**: Git, VS Code, 각종 유용한 도구들
